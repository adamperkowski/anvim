# anvim

my neovim config :3 (nix is certified crazy)

try it out with

```sh
nix run github:adamperkowski/anvim
```

## Features

- Up-to-date with latest neovim
- Fast af (think so at least)
- Full nix support
- Catppuccin :3
- LSP and stuff
- NO bloat
- Much more...
